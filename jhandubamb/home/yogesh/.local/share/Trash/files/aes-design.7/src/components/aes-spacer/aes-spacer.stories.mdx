# The Spacer

Wrap anything with padding. Use `aes-spacer` to avoid

1. Writing custom styles within a component just to add padding
1. Wrap entire components with configurable whitespace

By default, `aes-spacer` provides NO padding. Here is an `aes-icon` component with NO
padding on all 4 axes. Note the black background is the actual element we'll be adding
padding to.

<Canvas>
  <Story name="Default">
    {({ t, r, b, l }) =>
      html`
        <aes-spacer
          t="${t}"
          r="${r}"
          b="${b}"
          l="${l}"
          class="bg-neutrals-black"
        >
          <aes-icon icon="logoWhite" class="w-12"></aes-icon>
        </aes-spacer>
      `
    }
  </Story>
</Canvas>
