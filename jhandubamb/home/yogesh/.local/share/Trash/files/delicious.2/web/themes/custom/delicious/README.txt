INTRODUCTION
------------

 Our Kitchen is a Bartik based theme and design inspiration is taken from https://bootstrapmade.com/demo/Delicious/


 REQUIREMENTS
 ------------ 

 The following modules/theme are required:
 AddToAny  (https://www.drupal.org/project/addtoany)
 Wow-Animation (https://wowjs.uk/)
 FlexSlider (https://www.drupal.org/project/flexslider)
 Libraries (https://www.drupal.org/project/libraries)
 SuperFish (https://www.drupal.org/project/superfish)
 Slik Slider (https://www.drupal.org/project/slick)
 


 INSTALLATION
 ------------

1. Download and enable the required modules (Libraries (2.x)http://drupal.org/project/libraries)
2. Download and add flexslider libraries plugin under:

-- https://github.com/woocommerce/FlexSlider Rename the file to be located at :
    /web/libraries/flexslider/ 

3. Visit and download WOW-ANIMATION minified JavaScript and css file. Locate file to :
    web/themes/custom/Delicious/js     ---- js files
    ewb/themes/custom/delicious/css    ---- css files

4. Check /admin/reports/status to ensure the everything was properly working.

5. Download  Our Kitchen Theme and locate this in /theme folder.

CONFIGURATION
-------------
1.
go to /admin/appearance/settings/ and enable  our Kitchen theme.

2. Block settings
go to /admin/structure/block  and remove extra block which is not required

3. Frontpage View settings
go to /admin/structure/views/view/Frontpage and for slider visit /admin/structure/views/view/slider
and for managing different content view setting visit /admin/structure/views/viewbytag
